Calculator app

This project addresses positioning issues with flexbox, CSS pseudoclass :nth-child() and JavaScript Function() that allows us to evaluate a string-based expression.

The programming of the calculator is done by means of a finite state machine composed of three states: the initial state where the calculator values are at zero and a value is entered, the second state where several values are entered and the third state where the user presses the equals button.
